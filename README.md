## ຂໍ້ກຳນົດການອອກແບບ Responsive (Tailwind CSS)
ເພື່ອໃຫ້ເວັບໄຊສະແດງຜົນໄດ້ສວຍງາມ, ເຮົາຈະໃຊ້ Grid System ນີ້

## Responsive code : " grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 xl:grid-cols-5 gap-4 p-4 "
* ມືຖື (Mobile): ສະແດງ 1-2 ອັນຕໍ່ແຖວ 
    * ຄຳສັ່ງ: `grid-cols-2`
* ແທັບເລັດ (Tablet): ສະແດງ 3 ອັນ 
    * ຄຳສັ່ງ: `md:grid-cols-3`
* PC (Desktop): ສະແດງ 4-5 ອັນ 
    * ຄຳສັ່ງ: `lg:grid-cols-4` ຫຼື `xl:grid-cols-5`
    
+  Hamburger Menu
+  Dark Mode

## Interactive Features
* Product Quick View
* Hover Effects & Animation
* Real-time Cart Badge : ເວລາກົດ "ເພີ່ມລົງກະຕ່າ" ໃຫ້ຕົວເລກເທິງໄອຄອນກະຕ່າອັບເດດທັນທີ (ໃຊ້ JavaScript ຈັດການ) ແລະ ອາດຈະມີການແຈ້ງເຕືອນນ້ອຍໆ (Toast Notification) ເດັ້ງຂຶ້ນມາມຸມຈໍວ່າ "ເພີ່ມລົງກະຕ່າສຳເລັດແລ້ວ"
